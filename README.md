# Portfolio Next.js

Portfolio personnel de Mushota Raphael - Backend Developer

## Démarrage

Installer les dépendances :

```bash
npm install
```

Lancer le serveur de développement :

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

Pour créer une version de production :

```bash
npm run build
npm start
```

## Technologies

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Motion (Framer Motion)
- Lucide React (icônes)
